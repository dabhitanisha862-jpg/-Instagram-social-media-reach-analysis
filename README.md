Instagram Social Media Reach Analysis Dashboard
A dynamic and interactive Power BI Dashboard built to track, analyze, and visualize Instagram reach and engagement metrics. This project transforms raw social media data into actionable insights, helping creators or brands understand audience behavior and optimize their content strategy.
📊 Project Overview
Understanding audience growth and content performance is key to modern digital marketing. This dashboard provides a consolidated view of critical performance indicators (KPIs) for an Instagram account, focusing on how content reaches users and drives engagement.
The primary goal of this project was to clean, model, and visualize Instagram metrics to identify trends, top-performing content types, and audience demographics.
🔑 Key Features & Insights
•	Reach Analysis: Tracks impressions and follower vs. non-follower reach over time to evaluate organic growth.
•	Engagement Tracking: Monitors likes, comments, shares, and saves to see which posts resonate most with the audience.
•	Content Performance: Compares different media formats (Reels, Carousels, Single Images) to determine the most effective strategy.
•	Audience Demographics: Visualizes follower distribution by location, age group, and gender (if available in your dataset).
•	Interactive Filters: Users can slice and dice the data by date range, content category, or time of posting.
🛠Tech Stack & Skills Used
•	Business Intelligence Tool: Power BI Desktop
•	Data Transformation: Power Query (ETL process, handling missing values, changing data types)
•	Data Modeling: Star schema modeling with data/calendar tables
•	DAX Formulas: Created custom measures for key metrics like Engagement Rate, Average Reach per Post, and Month-over-Month Growth.
